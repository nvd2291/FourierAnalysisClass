# FourierAnalysisVisualizerTool

This tool can be used to create several different types of signals and plot the time domain and FFT signal. 

Available Signal Types:
 - Sine
 - Square
 - Sawtooth
 - Triangle

Available Noise Options:
  - White
  - Brown
  
  * To be added:
  - Pink
  - Blue
  - etc.
